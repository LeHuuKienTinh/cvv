DTUBE - ỨNG DỤNG XEM PHIM TRỰC TUYẾN TÍCH HỢP CHATBOT, MÃ QR

02/2025-05/2025

Đồ án tốt nghiệp - Đồ án nhóm
  • DTube là một nền tảng xem phim trực tuyến cho phép người dùng đăng ký tài khoản, lựa chọn gói dịch vụ trả phí, thanh toán trực tuyến và xem các bộ phim được phân loại theo thể loại. Hệ thống bao gồm tính năng quản lý người dùng, phim, gói cước, giao dịch,báo cáo vi phạm, và thanh toán tự động qua mã QR.
  • Công nghệ sử dụng: React, SCSS, Node.js, Express.js, MySQL, JWT, VietQR, SePay, RESTful API
  • Chức năng chính:
    • Đăng ký / Đăng nhập người dùng (JWT), đăng nhập bằng Token, đổi mật khẩu, quên mật khẩu
    • Xem phim, xem danh sách phim, tìm kiếm, lọc theo thể loại
    • Chức năng like phim, báo cáo tập phim lỗi
    • Đăng ký và thanh toán gói dịch vụ bằng QR code (SePay), xác thực realtime
    • Trang quản trị: quản lý người dùng, phim, báo cáo, gói cước, bình luận, giao dịch, thông báo
  • Vai trò: Phụ trách toàn bộ Frontend (ReactJS) và 1 phần Backend( xây dựng API, tích hợp thanh toán). Xây dựng giao diện người dùng bằng React và SCSS.
  • Link Github:
    • https://github.com/LeHuuKienTinh/dtube-frontend
    • https://github.com/LeHuuKienTinh/dtube-backend

DZFULLSTACK - WEBSITE THƯƠNG MẠI ĐIỆN TỬ 09/2023-02/2024

Dự án cá nhân
  • DzFullstack là nền tảng thương mại điện tử cho phép người dùng đăng ký, đăng nhập, xem sản phẩm, thêm vào giỏ hàng, đặt hàng và thanh toán online. Quản trị viên có thể quản lý sản phẩm, đơn hàng và người dùng qua dashboard riêng.
  • Công nghệ sử dụng: Vue, PHP Laravel, MySQL, VietQR, JWT,RESTful API
  • Chức năng chính:
    • Đăng ký/Đăng nhập người dùng(JWT), quên mật khẩu, đổi mật khẩu
    • Xem danh sách sản phẩm, danh mục sản phẩm,tìm kiếm, giỏ hàng, thanh toán
    • Trang ADMIN quản lý đại lý, đơn hàng, sản phẩm, nhân viên, danh mục, khách hàng, phân quyền
    • Trang đại lý quản lý đơn hàng, nhập kho, sản phẩm, thống kê, ...
  • Vai trò: Phát triển toàn bộ hệ thống từ frontend đến backend, thiết kế cơ sở dữ liệu, bảo mật API và triển khai tính năng mua hàng.
  • Link Github:
    • https://github.com/LeHuuKienTinh/DZTMDT

KTMOBILE- WEBSITE THƯƠNG MẠI ĐIỆN TỬ 10/2024-01/2025

Dự án cá nhân
  • KTMobile là website thương mại điện tử chuyên bán điện thoại và phụ kiện. Người dùng có thể xem danh sách sản phẩm, thêm vào giỏ hàng, đăng ký/đăng nhập và đặt hàng. Hệ thống có trang quản trị riêng cho admin để quản lý sản phẩm, đơn hàng và người dùng.
  • Công nghệ sử dụng: Vue.js, PHP Laravel, MySQL, RESTful API
  • Chức năng chính:
    • Đăng ký/Đăng nhập người dùng
    • Danh sách sản phẩm, tìm kiếm, lọc theo thương hiệu, giỏ hàng
    • Đặt hàng và quản lý đơn hàng
    • Trang quản trị: thêm/sửa/xóa sản phẩm, duyệt đơn hàng
    • Phân quyền người dùng (admin/user)
  • Vai trò: Phụ trách cả frontend (Vue.js) và backend (Laravel), thiết kế giao diện người dùng, xây dựng RESTful API và xử lý logic đặt hàng, quản trị sản phẩm.
  • Link Github:
    • https://github.com/LeHuuKienTinh/KT_DTDD
